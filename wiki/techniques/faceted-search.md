Interactive information access method where users progressively filter a result set by selecting values from multiple independent classification facets, each typically showing result counts; applied to RDF datasets, facets are derived from schema properties and classes, enabling exploration without SPARQL knowledge. Systems such as SemFacet extend faceted search with numeric ranges, aggregation, and reachability-based restrictions.
