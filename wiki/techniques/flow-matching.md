## Definition
A generative modeling technique that trains a neural network to approximate the **velocity field** $v_\theta(x_t,t)$ of an ODE that transports samples from a source distribution $p_0$ to a target data distribution $p_1$ via a smooth interpolant $x_t = I_t(x_0,x_1)$:

$$\mathcal{L}_{FM}(\theta) = \mathbb{E}_{t,x_0\sim p_0,x_1\sim p_1}\left\|v_\theta(x_t,t) - \frac{d}{dt}I_t(x_0,x_1)\right\|^2$$

A common linear interpolant is $I_t(x_0,x_1)=(1-t)x_0+tx_1$. Samples are generated by integrating the learned ODE from $t=0$ to $t=1$ using any ODE solver.

### Why it matters
Flow matching generalises diffusion by allowing non-Gaussian source distributions and provides a clean optimal-transport interpretation. It supports exact likelihood evaluation and can be optimized for straight-line (geodesic) flows that reduce the number of integration steps needed at inference time. It has become a leading alternative/complement to DDPM-style models in recent large-scale image and video generation systems.
