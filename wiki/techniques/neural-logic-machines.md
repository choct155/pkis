Neural Logic Machines (NLM) are a neurosymbolic architecture for inductive learning and logic reasoning that represents logic predicates as probabilistic tensors grounded over a set of objects, performing sequential logic deduction through layers that aggregate multi-arity relational information — enabling lifted rule learning (rules that generalize across objects) without exponential scaling in rule complexity.
