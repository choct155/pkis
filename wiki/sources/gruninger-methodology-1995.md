## Notes
Introduces competency questions (CQs) as the primary instrument for defining ontology scope. A CQ is a natural language query the ontology must answer; the CQ set defines what the ontology is for and provides testable completeness conditions. Directly maps to C(q) formalism: the query dependency distribution is the PKIS analog of the CQ set. Minimum concept node scope = content required to answer all CQs that depend on the concept.
