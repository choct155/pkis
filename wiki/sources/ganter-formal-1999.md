Canonical reference for FCA. Defines formal context K=(G,M,I), formal concept as (extent, intent) pair, concept lattice, and closure operators. Directly relevant to concept boundary finding: a concept is well-bounded when no attribute can be added without losing objects and no object can be added without losing attributes — the maximal rectangle criterion. Prerequisite-of edges have a lattice-theoretic grounding in subconcept ordering.
