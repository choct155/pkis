## Definition
Goedel's result that, once first-order logic is extended with the mathematical-induction schema for arithmetic, there exist true arithmetic sentences that cannot be proved from any recursively enumerable set of true axioms. The proof Goedel-numbers sentences and proofs so that number theory can name its own sentences, then constructs a sentence asserting its own unprovability — which must be true yet unprovable.
