## Definition
Two properties of a heuristic function h(n) that govern whether informed search is cost-optimal. A heuristic is **admissible** if it never overestimates the true cost to reach a goal: h(n) <= h*(n) for all n; an admissible heuristic is therefore optimistic. A heuristic is **consistent** (or monotone) if for every node n and every successor n' generated by action a, h(n) <= c(n,a,n') + h(n') -- a form of the **triangle inequality**. Every consistent heuristic is admissible, but not conversely. Consistency is equivalent to the f = g+h value being nondecreasing along every path, so A* with a consistent heuristic expands nodes in nondecreasing f-order and the first time it reaches any state it does so on an optimal path (no node need ever be re-added to the frontier). Admissibility alone guarantees A* returns a cost-optimal solution; consistency additionally guarantees A* (as a graph search) never re-expands a state. Heuristics derived from the exact solution cost of a relaxed problem or a subproblem are automatically admissible (and, being exact costs, consistent). Composite heuristics formed by max(h1,...,hk) of admissible (resp. consistent) heuristics are themselves admissible (resp. consistent).
