## Definition
The learning task in which an agent observes a training set of N input-output pairs (x_1,y_1),...,(x_N,y_N), each generated by an unknown function y=f(x), and must discover a hypothesis h drawn from a hypothesis space H that approximates f and generalizes to unseen inputs. The output y_i is the ground truth (the label). The task splits by output type: classification when the output is one of a finite set of values, regression when it is a number. This is one of the three feedback regimes of machine learning, contrasted with unsupervised learning (patterns without explicit feedback, e.g. clustering) and reinforcement learning (learning from rewards and punishments).

A hypothesis is consistent if h(x_i)=y_i for every training example; with continuous outputs one instead seeks a best-fit function whose h(x_i) is close to y_i. The true measure of a hypothesis is not training-set fit but performance on a held-out test set: h generalizes well if it accurately predicts test-set outputs. Choice of hypothesis space can be guided by prior knowledge, exploratory data analysis, or empirical comparison; under a probabilistic view one selects h* = argmax_h P(h|data) = argmax_h P(data|h)P(h), letting the prior P(h) encode a preference for simpler hypotheses (Ockham's razor).

Underpinning supervised learning is the stationarity assumption: future examples are drawn from the same fixed distribution as past ones, with examples independent and identically distributed (i.i.d.). Without it, no generalization guarantee is possible.
